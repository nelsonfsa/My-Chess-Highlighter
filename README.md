1. Get the files onto your computer

Option A (easy): On your GitHub repo page, click the green Code button, then Download ZIP. Unzip it.
Option B (git): Run git clone <your-repo-url> in a terminal.

2. Open Chrome's extensions page

Type chrome://extensions in the address bar and press Enter.
(Or go to menu ⋮ → Extensions → Manage Extensions.)

3. Turn on Developer mode

Flip the Developer mode toggle in the top-right corner. This unlocks the option to load your own extension.

4. Load the extension

Click Load unpacked (top-left).
Select the folder that directly contains manifest.json. (See note below.)

5. Confirm it loaded

Your extension now appears as a card on the page.
Click the puzzle-piece icon in the toolbar, then pin it for quick access.
